Мега-анализатор чатов

Откройте скрытые паттерны, темы и эмоции в вашей переписке

Загрузите историю чата

Как получить файл:

Telegram: Откройте чат в Telegram Desktop → Нажмите ⋮ → Export chat history → Снимите галочки с фото/видео → Убедитесь, что формат JSON → Export.

WhatsApp: Откройте чат на телефоне → Нажмите на имя → Export Chat → Выберите "Без медиафайлов" (Without media) → Сохраните полученный .txt или .zip файл.

Перетащите .json, .txt, .zip или зашифрованный .enc файл сюда
или нажмите для выбора

Расширенные настройки
Ускорение загрузки моделей (HuggingFace)

HuggingFace ограничивает скорость скачивания моделей без авторизации — загрузка может занять часы вместо минут. Введите бесплатный токен, чтобы ускорить процесс.
Получить токен →

hf_***...abcd
Умный анализ (Ollama LLM)

Требуется запущенный локально Ollama API (http://localhost:11434).

Работа на арендованном GPU (Vast.ai)

Вы можете зашифровать файл локально с помощью скрипта scripts/prepare_remote_upload.py и загрузить сюда полученный .enc файл.

⚠️ ВНИМАНИЕ: Шифрование защищает файл только при передаче и хранении на диске. Это НЕ ЗАЩИЩАЕТ от оператора сервера, который может намеренно инспектировать память процесса во время расшифровки. Используйте только проверенные хосты, если это критично.
